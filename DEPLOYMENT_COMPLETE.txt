📊 ENTROPY CATEGORIZATION FOR ALL FILE TYPES
═══════════════════════════════════════════════════════════════════════════════

ENTROPY RANGES PROVIDED FOR:

🟢 LOW ENTROPY (0.0 - 4.5) = READABLE/UNCOMPRESSED
   ✓ Empty/Uniform Data (0.0 - 1.5)
     • Blank files
     • Solid color images (1 pixel repeated)
     • Highly structured, repetitive data
   
   ✓ Uncompressed Text (1.5 - 3.5)
     • Plain text (.txt, .csv, .json, .md)
     • Source code (.py, .js, .java, .cpp, .c, .rb, .go)
     • Markup files (.xml, .html)
     • Log files
   
   ✓ Structured Data (3.5 - 5.0)
     • PDF documents
     • Office documents (DOCX, XLSX, PPTX, ODT, ODS, ODP)
     • Formatted documents
     • Database files (partial)

🟡 MEDIUM ENTROPY (4.5 - 6.5) = MIXED/COMPRESSED
   ✓ Mixed Content (5.0 - 6.0)
     • Photos (JPG, PNG, BMP)
     • Light compression archives
     • Documents with images
   
   ✓ Compressed/Binary (6.0 - 6.5)
     • Audio files (MP3, WAV, FLAC, AAC, OGG)
     • Video files (MP4, AVI, MKV, MOV, FLV)
     • ZIP archives with binary
     • Database files (complex)

🔴 HIGH ENTROPY (6.5 - 8.0) = COMPRESSED/ENCRYPTED
   ✓ High Compression (6.5 - 7.5)
     • Executable files (.exe, .dll, .elf, .so)
     • Packed executables
     • RAR archives
     • GZIP archives
     • 7-ZIP archives
   
   ✓ Maximum Entropy (7.5 - 8.0) = ENCRYPTED/OBFUSCATED
     • Encrypted files (.gpg, .aes, .encrypted)
     • Password-protected archives
     • Obfuscated executables
     • Random binary data
     • Steganography/data hiding
