==============================================================================
FORENSIC ANALYSIS GUIDE
==============================================================================

✅ LOW ENTROPY (< 4.0) = READABLE TEXT
   Actions:
   • Save for archival
   • Extract for indexing/searching
   • No security concerns
   Meaning: Plain text, source code, documents without encryption

⚠️  MEDIUM ENTROPY (4.0 - 6.0) = MIXED/FORMATTED CONTENT
   Actions:
   • Verify file type matches content
   • Check for metadata
   • Extract if archive
   Meaning: Photos, formatted documents, light compression

🔍 HIGH ENTROPY (6.0 - 7.5) = COMPRESSED OR BINARY
   Actions:
   • Check digital signature
   • Scan for malware
   • Verify source
   Meaning: Executables, compressed files, legitimate binary

⚠️  MAXIMUM ENTROPY (7.5 - 8.0) = ENCRYPTED/OBFUSCATED
   Actions:
   • 🚨 FLAG AS SUSPICIOUS
   • Request password/key
   • Analyze file structure
   • Check for malware signatures
   Meaning: Encrypted, heavily packed, or malicious obfuscation

FILE TYPE QUICK REFERENCE
==============================================================================

DOCUMENTS:
  .txt, .csv, .json, .xml, .html    → Entropy: 3.0 - 4.5 (LOW)
  .pdf, .docx, .xlsx, .pptx         → Entropy: 3.0 - 5.5 (MEDIUM-LOW)
  
SOURCE CODE:
  .py, .js, .java, .cpp, .c, .rb    → Entropy: 3.5 - 5.0 (MEDIUM-LOW)
  
IMAGES:
  Solid color                        → Entropy: 0.5 - 1.5 (VERY LOW)
  Photos (typical)                   → Entropy: 4.0 - 6.0 (MEDIUM)
  Noise/random                       → Entropy: 7.5 - 8.0 (MAXIMUM)
  
AUDIO/VIDEO:
  .mp3, .wav, .flac, .ogg           → Entropy: 6.0 - 7.5 (HIGH)
  .mp4, .avi, .mkv, .mov            → Entropy: 6.5 - 7.8 (MAXIMUM)
  
ARCHIVES:
  Text content (.zip, .tar.gz)      → Entropy: 2.5 - 4.5 (LOW)
  Binary content                     → Entropy: 5.0 - 7.0 (HIGH)
  Encrypted (.zip/.rar encrypted)    → Entropy: 7.5 - 8.0 (MAXIMUM)
  
EXECUTABLES:
  Uncompressed (.exe, .elf)          → Entropy: 6.5 - 7.5 (HIGH)
  Packed/obfuscated                  → Entropy: 7.0 - 8.0 (MAXIMUM)
  
DATABASE:
  .db, .sqlite, .mdb                → Entropy: 4.0 - 6.5 (MEDIUM to HIGH)

==============================================================================
MISMATCH DETECTION GUIDE
==============================================================================

⚠️  SUSPICIOUS PATTERNS:
   
1. File is .txt but entropy is 7.8
   → Likely encrypted text file
   → ACTION: Request key or use password recovery
   
2. File is .zip but entropy is 2.5
   → Archive with plain text, likely safe
   → ACTION: Extract and scan contents
   
3. File is .jpg but entropy is 7.9
   → Likely encrypted image or fake extension
   → ACTION: Verify magic bytes, check content
   
4. File is .exe but entropy is 3.5
   → Unlikely, script masquerading as executable
   → ACTION: Check magic bytes, analyze structure
   
5. File is .docx but entropy is 7.6
   → Encrypted or corrupted
   → ACTION: Request password or check integrity
